# Aplicación 2

Proyecto de aplicación de escritorio desarrollado en C# con Windows Forms. Esta es una aplicación cliente de interfaz gráfica que proporciona una ventana principal para la interacción del usuario.

## 📋 Descripción

Este proyecto es una aplicación Windows Forms construida en .NET Framework que incluye formularios personalizados y configuración de propiedades de la aplicación. La solución contiene componentes de interfaz gráfica reutilizables y configuración centralizada.
